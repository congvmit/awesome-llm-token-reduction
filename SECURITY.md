# Security Policy

## Supported Versions

We only support the latest version of the Awesome LLM Token Reduction list. If you find an issue with a specific link or entry, please report it via a Pull Request or Issue.

## Reporting a Vulnerability

If you discover a security vulnerability within this repository (e.g., a malicious link that was merged), please report it by opening a GitHub Issue with the label `security`.

We primarily curate links to external resources; we are not responsible for the security of those external tools. However, we strive to only include reputable and safe resources.
